# Kirarin

sparkle text and image :)

## Setup

```sh
pnpm add kirarin
yarn add kirarin
npm i kirarin
```

## Usage

```js
import React from 'react'
import { Kirarin, HeartYuraYura, HeartPowan } from 'kirarin'

~~~

<Kirarin>きらきら</Kirarin>
<HeartYuraYura>ゆらゆら</HeartYuraYura>
<HeartPowan>ぽわ〜ん</HeartPowan>
```
